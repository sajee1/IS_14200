Generate from SQL Screen
Screen Heading: WmDB > Service Generation > Generate from SQL
Use the Generate from SQL screen to create a service that performs a SQL statement that you specify.
To access this screen, click Service Generation in the WmDB Navigation panel; the Generate New Database Service Screen will be displayed. Click the Generate from SQL button. The server displays the Generate from SQL screen, which contains the following:
Evaluate
Opens the Input Binding Generation Screen, allowing you to identify the names and data types of the input parameters for the service you are generating. Click Evaluate after you have supplied the information on this screen.
Clear
Deletes any information in the Enter SQL Statement field and refreshes the screen.
Cancel
Exits the screen without creating the service and returns to the original screen.
Enter SQL Statement
Contains the SQL statement that you want the service to perform. You can specify a statement that is static or include either question marks (?) or template tags in the SQL statement to make it dynamic. The template tags you can use are the same as the tags you use in an output template. When you specify a dynamic SQL statement, the service expects input values to replace the question marks or template tags you specify.
Process webMethods template tags (%value, %ifvar, etc.) in this SQL statement
Enables the server o process the SQL statement when you include template tags in the SQL statement. You should select this checkbox if you include template tags in the SQL statement.
