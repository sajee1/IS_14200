Change Password Screen
Screen Heading: WmDB > Alias Management > Change Password
Use the Change Password screen to change the password for the database alias.
To access this screen, select Alias Management in the WmDB Navigation panel. Select a data source and click Edit. From the Edit Alias Information Screen, click Change password in the DB Password row.
New Password
New password you want to associate with the database alias.
Confirm Password
Enter the same password again to make sure you typed it correctly.
