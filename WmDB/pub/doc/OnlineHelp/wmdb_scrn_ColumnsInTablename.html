Columns in (TableName) Screen
Screen Heading: WmDB > Service Generation > Connection Parameters > Tables > Columns
Use the Columns in (TableName) screen to identify the database operation you want the service to perform and the columns on which you want the service to act.
To access this screen, open the Tables in (DataSourceName) Screen. Click the name of the table that you want the service to access; the server displays the (TableName) screen, which contains the following:
Generate SQL
Click this button after you have made your selections on this screen. The server generates the SQL for the service and displays the Input Binding Generation Screen, allowing you to identify the names and data types of the input parameters for the service you are generating.
Service type (select columns below)
Use the fields in this section to select the database operation that you want the service to perform.
Use this field...
To...
Select
Indicate the service is to perform the SELECT database operation. The service expects input values for the columns you select in the Columns section of the screen. The service selects the rows that match the input values specified for the selected columns.
Insert
Indicate the service is to perform the INSERT database operation. The service expects input values for the columns you select in the Columns section of the screen. The service populates the columns in the inserted rows using the input values specified for the selected columns.
Delete
Indicate the service is to perform the DELETE database operation. The service expects input values for the columns you select in the Columns section of the screen. The service selects rows that have the columns that match the input values specified for the selected columns, and deletes the selected rows.
Update
Indicate the service is to perform the UPDATE database operation. When you select Update, the server displays Set and Criteria fields in the Columns section of the screen. The service expects input values for the columns you select for both the Set and Criteria. The service selects rows that have columns that match the input values specified for Criteria. The service sets the value of all selected rows to the input value specified for Set.
Columns in Tablename
Use the fields in this section to select the columns on which the service will act. The server displays information about the data in each column.
Use this field to...
To...
Use this column
Select table columns on which the service will act. This field is displayed when you select the Select, Insert, or Delete database operation. The service expects input values for each column you select.
Set
Select table columns that the service will update for selected rows. This field is displayed only when you select the Update database operation. The service expects input values for each column you select.
Criteria
Select table columns that the service will use to select rows to update. This field is displayed only when you select the Update database operation. The service expects input values for each column you select.
Name
Display the name of the column.
Data Type
Display the SQL type for column.
Nullable?
Display whether the column allows null values.
Test SQL Queries
Use the field in this section to submit a SQL statement to the database. Type the SQL statement in the box, then click Execute query.
Use this field...
To...
Execute query
Send the SQL statement you specify to the database for execution. The server displays the results of the query.
